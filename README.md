# 📈 Stock Analyzer (Stacked LSTM)

A professional financial dashboard that uses Deep Learning to predict stock market trends. 

## ✨ Key Features
- **Stacked LSTM Model:** Uses multiple LSTM layers for complex pattern recognition.
- **10+ Technical Indicators:** Includes RSI, MACD, SMA, and more.
- **Interactive UI:** Built with Streamlit for real-time analysis.
- **Dynamic Parameters:** Adjustable lookback windows and buy thresholds.

## 🛠️ Tech Stack
- **Python** (3.11)
- **TensorFlow/Keras** (AI Model)
- **Streamlit** (Dashboard)
- **YFinance** (Live Market Data)
- **Plotly** (Interactive Charts)
